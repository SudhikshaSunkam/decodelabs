# Movie Review Sentiment Analysis using NLP

## Project Overview
This project performs Sentiment Analysis on movie reviews using Natural Language Processing (NLP) and Machine Learning. The model classifies reviews as either Positive or Negative based on their content.

## Dataset
IMDb Movie Reviews Dataset (50,000 reviews)

Features:
- review: Movie review text
- sentiment: Positive or Negative label

## Technologies Used
- Python
- Pandas
- NLTK
- Scikit-learn
- TF-IDF Vectorization
- Naive Bayes Classifier

## Project Workflow
1. Load the dataset.
2. Clean and preprocess text data.
3. Remove stopwords and perform lemmatization.
4. Convert text into numerical features using TF-IDF.
5. Split data into training and testing sets.
6. Train a Naive Bayes classifier.
7. Evaluate model performance.
8. Predict sentiment for new reviews.

## Results
The model achieved approximately 85.6% accuracy in classifying movie reviews as Positive or Negative.
